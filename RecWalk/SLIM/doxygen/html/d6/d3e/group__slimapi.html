Parameters
nrowsnumber of rows in the matrix
rowptr[rowptr[i], rowptr[i+1]) points to the indices in rowind and rowval
rowindcontains the column indices of the non-zero elements in matrix.
rowvalcontains the values of the non-zero elements in matrix.
ioptionsinteger or boolean options to the estimation routine.
doptionsdouble options to the estimation routine.
imodelif not null then it is pointer to model that will be used for initialization.
r_statusset to 1 on success
Returns
sparse representation of model in CSR format.
